# Calculator Project

## :page_with_curl: About
This is a functional calculator with standard arithmetic operations and a clean UI (an imitation of the iOS calculator).
This was a project I created while learning JavaScript foundations with Frontend Masters.

## :telescope: How To View
Open in your browser with [this link](https://sarah-dahan.github.io/calculator/).

## :computer: Technologies

- CSS
- JavaScript
